Hello, this is a pet (demonstrative) site of Gralevskyi Oleksandr.

As for me the fastest way to learn foreign lexicon is learning the most frequently used words in sphere you are interested in.

And this site can help you in such approach. You can take any text you are interested in and parse it into words. In the result of parsing you receive a number of times a word is occured in the text. You can save your result of parsing and after that delete unnecessary words. In such way you can form lists of words for learning.

The back end (server) site was created using Java 8 and Spring framework. The code is on git hub.

The front end is running using angular js. The code is also located on git hub.
